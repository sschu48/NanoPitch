# Slide Visuals

Editable SVG visuals for the 6-slide NanoPitch presentation.

Use these directly in slides or export them to PNG:

| File | Suggested slide | Purpose |
|---|---|---|
| `01-four-axis-card-grid.svg` | What It Does | Shows the four feedback axes as report cards. |
| `02-system-pipeline.svg` | How It Works | Shows the record -> browser model -> technique API pipeline. |
| `03-results-bars.svg` | Results | Shows the main VAD/noise augmentation win and validation snapshot. |
| `04-technique-reality-check.svg` | Limitations & Next Steps | Explains the current technique model and what data is needed next. |

All visuals are 1600x900 SVGs designed for 16:9 slides.
